# 📊 Swiggy Funnel Conversion Analysis

A data analytics case study that examines customer behavior across the Swiggy ordering funnel using **Microsoft Excel**. The project focuses on identifying conversion bottlenecks, analyzing user drop-offs, and generating actionable business insights through funnel metrics and dashboard analysis.

> **Note:** This repository contains an enhanced version of a university data analytics project developed for academic and portfolio purposes.

---

## Project Objectives

- Analyze customer movement across the ordering funnel.
- Calculate stage-wise conversion rates.
- Identify major user drop-off points.
- Perform Week-on-Week (WoW) analysis.
- Generate business insights using Excel dashboards.

---

## Dataset

The dataset contains daily session-level records for the customer journey, including:

- Listing Sessions
- Menu Sessions
- Cart Sessions
- Payment Sessions
- Order Sessions

**Dataset Source:** University case study dataset (academic use). :contentReference[oaicite:0]{index=0}

---

## Tools Used

- Microsoft Excel
- Pivot Tables & Charts
- Excel Formulas
- Dashboard Visualization

---

## Key Insights

- Overall conversion rate is approximately **5.4%**.
- **Menu → Cart (M2C)** is the largest drop-off stage.
- **Payment → Order (P2O)** has the highest conversion.
- Week-on-Week analysis helps identify genuine performance trends by removing weekday bias. :contentReference[oaicite:1]{index=1}
